思路二

如果遇到Cookie:admin-token=未知
如果遇到一个后台登录页面，抓包发现一个error返回参数为1，然后修改参数0，进行尝试发送后，遇到返回数据中有Cookie:admin-token=未知后，我们的思路是：

  1. 找小程序的token/Cookie，抓取过去进行尝试
  2. 找其他系统的弱口令进行尝试修改。
