变量与常量的区别

1
2
3
4

定义⽅式：常量使⽤ const，变量使⽤ var 或 :=。
值的修改：常量不可修改，变量可修改。
内存分配：常量在编译时确定，不占⽤运⾏时内存；变量在运⾏时分配内存。
使⽤场景：常量⽤于定义不会改变的值，变量⽤于定义可能会改变的值。

package main  
  
import "fmt"  
  
func adder() func(int) int {  
    sum := 0  
    return func(x int) int {  
       sum += x  
       return sum  
    }  
}  
func main() {  
    pos, neg := adder(), adder()  
    for i := 0; i < 10; i++ {  
       fmt.Println(pos(i), neg(-2*i))  
    }}
